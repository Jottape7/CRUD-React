# CRUD de Produtos

Sistema desenvolvido em React.js para cadastro, edição, listagem e exclusão de produtos.

## Tecnologias
- React.js
- JavaScript
- CSS
- Vite

## Funcionalidades
- Cadastro de produtos
- Listagem de produtos
- Edição de produtos
- Exclusão de produtos

## Como executar
cd crud
npm run dev
