# CRUD de Produtos

Sistema desenvolvido em React.js para cadastro, edição, listagem e exclusão de produtos.

## Tecnologias
- React.js
- JavaScript
- CSS

## Funcionalidades
- Cadastro de produtos
- Listagem de produtos
- Edição de produtos
- Exclusão de produtos

## Como executar
npm install
npm run dev
